First Project

I'm a big fan of project-based learning; nothing helps me internalize a lesson like utilizing it. I'm also a big fat geek, so Project 1 is going to be a character generator for the World of Darkness system.

Basic Info

Stack: I'm using this project to dust off my web skills, so I'll be using the Triforce (HTML, CSS, and Javascript) for coding, with an Angular framework.

Goal: A program that will let me quickly and easily build PC and NPC characters for my home game.

Milestones: (of course, these are subject to change)

  • Compile without error
  • Generate a Human character
  • Generate a Supernatural character
  • Generate a character using the WoD 1.0 setting and the WoD 2.0 mechanics
  • Generate an experienced character
  • Randomly generate a character

Requirements

  • Generate a splash screen
  • Character Creation steps
    1. Choose Background (Name, Concept, Splat, Virtue/Vice)
    2. Prioritize and select Attributes
    3. Prioritize and select Skills
    4. Select Skill Specialties
    5. Add Supernatural template
    6. Determine Advantages
    7. Select Merits
  • Generate character sheet
  • Add experience
  • Random generation
